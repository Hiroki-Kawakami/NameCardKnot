# NameCardKnotの使い方

NameCardKnotは、M5PaperまたはPaperS3に名札画像を表示し、端末同士で名札データを共有できるESP32 / ESP32-S3向けファームウェアです。

## 必要なもの

- [M5Paper](https://docs.m5stack.com/ja/core/m5paper)または[PaperS3](https://docs.m5stack.com/ja/core/PaperS3)
- FAT32でフォーマットしたmicroSDカード
  - 自分の名札データの読み込みと、相手から受信した名札データの保存に使用します
  - microSDカードの種類や状態によっては、動作が不安定になる場合があります

## 表示・共有する名札データの用意

[NameCardKnot Editor](https://hiroki-kawakami.github.io/NameCardKnot/)で名前や画像などを設定し、出力された `.mnc.pdf` ファイルをmicroSDカードへコピーしてください。

設定できる項目は以下のとおりです。

- **名前:** 共有データに含める名前
- **URL（任意）:** URLを1つ共有データに含められます。受信相手が確認できるほか、自分の端末でもQRコードとして表示できます
- **表示用画像:** 自分の端末に表示する画像。M5Paper / PaperS3の表示領域に合わせて、最大540×960ピクセルに変換されます
- **共有用画像1:** 共有相手へ送信する画像。表示用画像と同じ画像を使用することもできます
- **共有用画像2（任意）:** 共有相手へ送信する2枚目の画像
- **共有メッセージ（任意）:** 共有データに含めるメッセージ。受信相手が確認できます

共有用画像は最大405×720ピクセルに変換され、元の画像ファイルは共有データに含まれません。

出力された `.mnc.pdf` ファイルは、一般的なPDFビューアーで内容を確認できます。1ページ目が端末に表示する画像、2ページ目が共有される内容の確認用ページです。

## ファームウェアの書き込みと初期設定

M5Burnerから、お使いの機種に対応するNameCardKnotのファームウェアをダウンロードし、端末へ書き込んでください。

初回起動時は、画面の案内に従って言語と日時を設定します。日時は、受信した名札データの保存日時を記録するために使用します。

> [!WARNING]
> RTC（時計用IC）に誤った日時が有効な値として残っていると、初回起動時でも日時設定画面が表示されないことがあります。
>
> 起動後に画面左上の日付を確認し、ずれている場合は設定画面から日時を設定し直してください。

名札データはmicroSDカードから直接開くこともできますが、よく使用する名札は、ホーム画面の「自分の名札」欄から本体へインポートすることをおすすめします。

## 自動電源オフと復帰

消費電力を抑えるため、無操作の状態が一定時間続くと、自動的に端末の電源が切れます。

- 名札の表示中: 1分
- それ以外の画面: 5分

電子ペーパーの特性により、電源が切れた後も画面の表示は残ります。復帰方法は機種によって異なります。

- **M5Paper:** 側面のボタンを、名札表示のメニューまたはホーム画面が表示されるまで約3秒間長押しします
- **PaperS3:** 側面のボタンを1回押し、名札表示のメニューまたはホーム画面が表示されるまで待ちます。長押しやダブルクリックはしないでください

## 名札データの共有・交換

送信側は「共有」画面、受信側は「受信」画面を開きます。端末の画面同士を重ねるように近づけ、そのまま約2秒間保持してください。

PaperS3では、画面を離してよいタイミングでブザー音が鳴ります。画面を離した後も名札データの転送は続くため、転送が完了して端末が再起動するまで、そのままお待ちください。

通常は共有側から受信側へ名札データが送信されます。次の両方を有効にすると、お互いの名札データを送信する交換動作になります。

- 共有側の「相手からの返信を許可する」
- 受信側の「自分の名札データを返信する」

受信した名札データはmicroSDカードの `ReceivedCards` フォルダーに保存され、ホーム画面の「ギャラリー」から確認できます。

## 設定

設定画面では、以下の項目を変更できます。

- **日時:** 日付と時刻を設定します
- **言語:** UIの表示言語を日本語または英語に切り替えます
- **画面を反転:** 画面の表示方向を180度回転します
